# ML Project 1: Girl_power team 

This project is about predicting coronary heart disease (MICHD) for an individual based on the Behavioral Risk Factor Surveillance System (BRFSS) dataset. 
Our files contain the following code:

- implmentations.py: contains all the machine learning models used for the project, as well as some helper functions for those models. 
- helpers.py: contains methods used to load the dataset, create a submission etc
- dataprocessing.ipynb: our main notebook, it contains the pipeline for data processing.


Please run preprocessing before running run.ipynb
